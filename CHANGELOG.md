## v1.0.0 (2025-01-01)

### 🎉 初始版本发布

#### ✨ 新增功能
- 📝 **单份作业批改** - 支持上传单张作业图片进行智能批改
- 📚 **批量作业批改** - 支持同时批改多份学生作业
- 🎯 **多种题型支持** - 支持选择题、填空题、计算题、作文题
- 📊 **详细批改结果** - 提供逐题评分、解析说明和总体反馈
- 🎨 **多科目支持** - 支持数学、语文、英语、物理、化学、地理、历史、生物等科目
- 🔍 **图片格式验证** - 自动验证和压缩上传的作业图片
- 📝 **结构化输出** - 按照标准JSON格式返回批改结果
- 🚀 **MCP协议支持** - 完整兼容Model Context Protocol规范

#### 🔧 技术特性
- **TypeScript开发** - 完整的类型定义和类型安全
- **模块化架构** - 清晰的服务分层和工具分离
- **错误处理** - 完善的异常捕获和错误提示
- **日志系统** - 分级别的日志记录功能
- **环境配置** - 灵活的环境变量管理
- **图片处理** - Base64图片验证和压缩功能

#### 📋 支持的工具
- `grade_homework` - 智能批改单份作业
- `batch_grade_homework` - 批量批改多份作业

#### 🎯 评分标准
- **A级 (90-100%)** - 优秀表现
- **B级 (80-89%)** - 良好表现  
- **C级 (70-79%)** - 中等表现
- **D级 (60-69%)** - 及格表现
- **F级 (<60%)** - 需要改进

#### 🔌 集成支持
- **VS Code + Cline** - 完整配置示例
- **Claude Desktop** - 详细配置说明
- **其他MCP客户端** - 标准MCP协议兼容
